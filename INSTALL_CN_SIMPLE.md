# 📦 TeachAny 一键安装指南（最简单）

> **适用对象**：国内中小学教师，无需翻墙，无需下载，**只需一个链接**

---

## ⚡ 推荐安装（下载 Release ZIP，秒装）

### 方法 A：下载 Release ZIP

1. 打开：https://github.com/weponusa/teachany/releases/latest
   （无法访问 GitHub？请用代理或 VPN，Gitee 暂不支持 Release 下载）
2. 下载 **`teachany-skill-standard-*.zip`**（约 **37MB**，解压后 **117MB**，含全部地图资产）
3. 解压到 `~/.agents/skills/teachany/`（覆盖已有目录）
4. 重启 CodeBuddy/WorkBuddy

### 📏 档位对照

| 档位 | 安装方式 | 大小 | 含什么 | 适用场景 |
|---|---|---|---|---|
| **standard** ⭐ | sparse checkout | **~140MB** | 制作器 + 知识树 + 课标摘录 + 全部地图（历史/政区/自然地理），不含既有课件 | 所有学科课件制作 |
| ~~full~~ | sparse-checkout disable | ~~1.7GB~~ | + 328 社区课件 + 18 示例 | ⚠️ 不推荐，通过 Gallery 在线浏览即可 |

> 💡 **Hero 图、社区课件、情境插图通过 CDN 按需加载**，不占用本地空间。选 **standard** 即可覆盖所有学科场景。
> 
> ⛔ **既有课件不下载到本地**：328 个成品课件已迁移到 `weponusa/teachany-courseware` 仓库的 `community/` 目录中，用户通过在线 Gallery 浏览和学习，不需要下载到本地。安装 skill 的目的是**制作新课件**，不是收藏旧课件。

---

## 🔧 方法 B：原有 `/install-skill` 路径（普通 clone 约 220MB）

> ✅ **已优化（v7.9.13）**：课件资产已迁移到独立仓库 `weponusa/teachany-courseware`，安装源只保留制作器核心（规范 + 脚本 + 知识树 + 课标摘录 + 全部地图）。

### 方法：复制链接 → 粘贴到 CodeBuddy

**安装链接**（GitHub 主仓，已完成轻量化）：
```
https://github.com/weponusa/teachany
```

> 注：Gitee 镜像因历史大对象超过配额，待仓库 GC 完成后再恢复为国内推荐源。

### 安装步骤（30秒完成）

1. **复制上面的链接**（Ctrl+C 或 Command+C）

2. **打开 CodeBuddy/WorkBuddy 应用**

3. **输入安装命令**：
   ```
   /install-skill https://github.com/weponusa/teachany
   ```
   或者在 **Skills 管理** 界面：
   - 点击 **"从 URL 安装"**
   - 粘贴链接
   - 点击 **"安装"**

4. **等待安装完成**（普通 clone 约 220MB，视网络情况约 1-3 分钟）

5. **开始使用**：
   ```
   /teachany 请生成"一次函数的应用"的课件
   ```

## 🆘 遇到问题？

### 问题1：安装失败，提示"网络错误"

**解决方案**：
1. 检查网络连接是否正常
2. 关闭 VPN/代理
3. 尝试切换到备用链接
4. 联系管理员检查防火墙设置

### 问题2：安装后找不到 Skill

**解决方案**：
1. 重启 CodeBuddy/WorkBuddy 应用
2. 在 Skills 管理中点击 **"刷新列表"**
3. 检查安装日志是否有错误提示

### 问题3：如何卸载？

在 Skills 管理界面：
1. 找到 **"TeachAny"**
2. 点击右侧的 **"卸载"** 按钮
3. 确认删除

## 🎓 教育理念

TeachAny 的核心理念：

1. **真实场景驱动**：每个知识点都对应现实生活中的应用
2. **五镜头教学法**：提问→定义→拆解→类比→应用
3. **ABT 叙事框架**：背景（AND）→冲突（BUT）→结论（THEREFORE）
4. **课标对齐**：所有内容对齐 2022 版课程标准

**我们相信**：教育不是填鸭式灌输，而是激发学生的好奇心，帮助他们建立知识与现实的连接。
